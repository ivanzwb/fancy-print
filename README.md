## 硬件路线（ZINK）

打印技术已定位为 **ZINK 全彩无墨**，耗材话术统一为 **ZINK 纸**（可涂色、剪纸、换装的手工向，非相纸洗照片）。**ZINK 路线、样机与幅面口径**见 [`doc/端侧软件与工程样机技术分析.md`](doc/端侧软件与工程样机技术分析.md#demo-kit-bom)（**§10** 起：工程样机 BOM / Bring-up；**§1～§9** 量产 OS 与端上应用）。
