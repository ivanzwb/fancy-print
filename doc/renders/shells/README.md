# 可更换主题外壳 — 渲染 / 摄影素材

本目录存放 **仅外壳（B 层）** 或与 **基础款整机基准图** [`../../fancy-print-product-render.png`](../../fancy-print-product-render.png) 的合成成品。

## 命名建议

- `shell-{主题}-v{版本}.png`  
  例：`shell-rabbit-v1.png`、`shell-spring-festival-v1.png`

## 约束（摘录自 [`../../product-render-system.md`](../../product-render-system.md)）

- 不改动 **基准图上的机身比例**（屏、出纸口、按键位）。  
- 外壳覆盖范围以 **打印仓上盖区域** 为主；若需全包，须与结构确认 **拆装路径与热设计**。
